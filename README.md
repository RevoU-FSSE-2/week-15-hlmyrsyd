## Hello!
This is an app that able to test CORS using typescript.

## DEPENDENCIES
- Node, Express
- Typescript
- CORS
- helmet
- morgan
- nodemon
- UUID

## HOW TO USE

### Clone this repo
`
git clone https://github.com/RevoU-FSSE-2/week-15-hlmyrsyd.git
`

### Install dependencies
`
npm install
`

### Start the script
`
npm start
`
